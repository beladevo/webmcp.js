# Security Policy

webmcp.js is pre-1.0 software for an experimental browser capability. Treat registered tools as executable application behavior.

## Reporting

Do not publish exploit details in a public issue. Open a minimal issue asking for a security contact, or use the repository security advisory flow if it is enabled.

## Supported Versions

Security fixes target the latest published version.

## Application Requirements

Applications using webmcp.js should validate tool input, require approval for high-impact actions, enforce application authentication for user-specific actions, and avoid logging sensitive data.
